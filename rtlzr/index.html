JavaScript Rules Labs :: RTLzr

Flips any page direction from LTR to RTL or RTL to LTR and export the CSS changes. Useful when localizing applications for RTL direction languages such as Arabic, Hebrew, Persian, etc.

Install it from Mozilla Add-ons or Download Page

GitHub source code
